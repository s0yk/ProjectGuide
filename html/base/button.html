baseButton Guide

버튼 스타일 리스트입니다.

각 유형별로 나열되어 있으며 클래스를 이용해 스타일을 변형한다.

버튼의 기본 클래스는 btn이며, 크기 조절 시 btn-sm으로 조절한다.
스타일은 btn-basic과 같은 클래스를 추가로 사용한다.

예시) 기본 스타일의 작은 버튼 생성 class="btn btn-sm btn-basic"

- Basic Button btn-basic
- Primary Button btn-pri
- Icon Button Left btn-ico-lt
- Icon Button Right btn-ico-rt
